1. 
 J'ai fait ma Dataviz et mes graphiques Plotly en allant fouiller dans la documentation officielle de Plotly. 
Pour m'aider sur la structure technique, je suis aussi allé sur des sites comme W3Schools,
ce qui m'a permis de coder le graphique 1 (l'histogramme) et le graphique 3 (les barres horizontales).




2.
 Pour que ma Dataviz raconte une vraie histoire, j'ai utilisé des articles de :

-France Musique

-Le Monde Pixels

-SNEP

-Slate

Grâce à ces sites, j'ai pu comprendre comment fonctionne l'algorithme que Spotify cache derrière son application. 
Ça m'a aussi permis d'ajouter des analyses plus "scientifiques", comme l'effet du tempo sur le cerveau humain. 
C'est ce qui m'a donné tout le contenu informatif du site.



3. Le rôle de l'IA Pour le code, je me suis servi de l'IA pour deux choses précises :

Le nettoyage : Elle m'a conseillé d'utiliser PapaParse pour traiter ma base de données (BDD).
 Même si j'ai écrit le code moi-même avec la doc, l'IA m'a aidé à bien comprendre ce que la doc racontait.

La correction d'erreurs : Elle m'a aidé à comprendre mes bugs. 
Par exemple, sur le graphique 4, ma moyenne n'était pas bonne parce que mon résultat ne tombait pas sur 100%.
L'IA m'a expliqué pourquoi mon calcul ne marchait pas et comment le résoudre pour avoir un résultat cohérent.
